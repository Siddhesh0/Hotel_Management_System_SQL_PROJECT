📌 Project Overview

This project simulates a real hotel management environment where multiple hotel operations are managed through SQL. It includes handling customer records, room bookings, payment tracking, staff management, and analytical reporting using structured SQL queries.

The project is ideal for practicing advanced SQL concepts through practical business scenarios.

🛠️ Technologies Used
SQL
MySQL / PostgreSQL
Relational Database Management System (RDBMS)
📂 Database Modules
✅ Customer Management
Customer registration and details
City-wise customer analysis
Customer booking history
✅ Room Management
Room type and pricing management
Capacity-based room analysis
Premium and budget room filtering
✅ Booking Management
Booking creation and tracking
Check-in and Check-out analysis
Revenue and occupancy insights
✅ Payment Management
Payment method tracking
High-value and low-value payment analysis
Online and card payment reports
✅ Staff Management
Staff roles and contact management
Department and role-based analysis
Staff performance reporting
